Dead simple script to setup my new Mac:
```shell
cd ~/Downloads
curl -O https://raw.githubusercontent.com/Dewwi/mac-setup-script/master/bash_prompt
curl -sL https://raw.githubusercontent.com/Dewwi/mac-setup-script/master/defaults.sh | bash
curl -O https://raw.githubusercontent.com/Dewwi/mac-setup-script/master/install.sh
chmod +x install.sh
./install.sh
```
